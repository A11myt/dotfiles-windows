# VSCode Keybindings (vim plugin)

Leader: `Space`

## Dateien

| Binding | Aktion |
|---------|--------|
| `<Leader>w` | Speichern |
| `<Leader>q` | Tab schließen |
| `<Leader>e` | Explorer toggle |

## Suche (Telescope-Äquivalent)

| Binding | Aktion |
|---------|--------|
| `<Leader>ff` | Dateien suchen (QuickOpen) |
| `<Leader>fg` | Text in Dateien suchen |
| `<Leader>fb` | Offene Buffer/Tabs |
| `<Leader>fh` | Befehlspalette |

## Buffer-Navigation

| Binding | Aktion |
|---------|--------|
| `]b` | Nächster Tab |
| `[b` | Vorheriger Tab |
| `<Leader>bd` | Tab schließen |

## Splits

| Binding | Aktion |
|---------|--------|
| `<Leader>sv` | Split rechts |
| `<Leader>sh` | Split unten |
| `<Leader>se` | Splits angleichen |
| `<Leader>sx` | Split schließen |

## LSP

| Binding | Aktion |
|---------|--------|
| `gd` | Definition |
| `gD` | Deklaration |
| `gr` | Referenzen |
| `gi` | Implementierung |
| `gh` | Hover-Info |

## Formatierung & TypeScript

| Binding | Aktion |
|---------|--------|
| `<Leader>mp` | Datei formatieren |
| `<Leader>to` | Imports sortieren |
| `<Leader>ti` | Fehlende Imports ergänzen |
| `<Leader>tr` | Datei umbenennen |

## Terminal & Copilot

| Binding | Aktion |
|---------|--------|
| `<Leader>tt` | Terminal toggle |
| `<Leader>cc` | Copilot Chat öffnen |
| `<Leader>cx` | Copilot Chat leeren |
| `<Leader>cl` | `console.log()` einfügen (normal) |
| `<Leader>cl` | `console.log(auswahl)` (visual) |
