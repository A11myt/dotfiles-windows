# Windows Terminal Keybindings (tmux-Ersatz)

## Tabs

| Shortcut | Aktion |
|----------|--------|
| `Ctrl+T` | Neuer Tab |
| `Ctrl+Shift+K` | Tab / Pane schließen |

## Panes (Split)

| Shortcut | Aktion |
|----------|--------|
| `Alt+Shift+V` | Split vertikal (rechts) |
| `Alt+Shift+S` | Split horizontal (unten) |
| `Alt+Shift+D` | Pane duplizieren (auto) |

## Pane-Navigation (vim-style)

| Shortcut | Aktion |
|----------|--------|
| `Ctrl+Alt+H` | Fokus links |
| `Ctrl+Alt+J` | Fokus unten |
| `Ctrl+Alt+K` | Fokus oben |
| `Ctrl+Alt+L` | Fokus rechts |

## Pane-Größe

| Shortcut | Aktion |
|----------|--------|
| `Ctrl+Shift+←` | Kleiner (links) |
| `Ctrl+Shift+↓` | Kleiner (unten) |
| `Ctrl+Shift+↑` | Größer (oben) |
| `Ctrl+Shift+→` | Größer (rechts) |

## Scrollen

| Shortcut | Aktion |
|----------|--------|
| `Ctrl+Shift+PgUp` | Seite hoch |
| `Ctrl+Shift+PgDn` | Seite runter |

## Sonstiges

| Shortcut | Aktion |
|----------|--------|
| `Ctrl+C` | Kopieren |
| `Ctrl+V` | Einfügen |
| `Ctrl+Shift+F` | Suchen |
